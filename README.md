# quantum-experiments

Python experiments with quantum computing algorithms using the QPanda3 quantum cloud framework.

## Algorithms

- **Grover's algorithm** — quantum search on 2 qubits via QPanda3 cloud backend
- **Quantum random number generation** — true randomness via quantum superposition
- **Algorithm comparison** — classical vs quantum approach benchmarks

## Tech Stack

- **Python**
- **QPanda3** — quantum cloud SDK (Origin Quantum)

| File | Description |
|------|-------------|
| `grover.py` | Grover's search algorithm implementation |
| `quantum_random.py` | Quantum random number generator |
